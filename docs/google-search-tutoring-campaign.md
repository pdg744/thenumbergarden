# Google Search Campaign: Tutoring and Family Visits

This is the initial Google Search campaign brief for The Number Garden's tutoring and family-visit funnel.

The campaign is a small local search experiment. Its job is to learn which parent search intents produce qualified family visits, not to maximize traffic.

## Campaign Setup

Campaign name:

- `Search | Family Visits | Portland`

Primary conversion:

- `book_appointment`

Secondary observed conversions:

- `newsletter_signup_submit`
- `camp_interest_signup_submit`

Landing-page plan:

- Primary landing page: `/family-visit/`
- Search-intent landing page to test later: `/portland-math-tutor/`

Location targeting:

- Start with a tight radius around `1717 NE 42nd Ave`
- Alternative: target selected nearby ZIP codes
- Use people in or regularly in the targeted location when available

Network:

- Google Search
- Search partners off for the first learning period, unless volume is too low

Match types:

- Start with exact and phrase match
- Do not use broad match until search-term quality is understood

Bidding:

- Start conservatively
- Optimize for booked appointments once conversion tracking is confirmed
- Do not optimize for newsletter signups unless appointment volume is too low to learn from

## Asset Standard

Google responsive search ads allow up to 15 headlines and 4 descriptions. Headlines are limited to 30 characters, descriptions to 90 characters, and path fields to 15 characters each. Google recommends multiple headline and description options so the system can test combinations, and recommends at least two responsive search ads per ad group when possible.

For this first small campaign:

- create 3 tightly themed ad groups
- create 1 complete responsive search ad per ad group first
- each RSA should include 12-15 headlines
- each RSA should include 4 descriptions
- avoid pinning unless a message must always appear
- add at least 4 sitelinks with descriptions
- add 6-10 callouts
- add 1 structured snippet set

Once search terms and conversion quality are clearer, add a second RSA per ad group with a distinct message angle.

## Review Criteria

Every headline and description should pass these filters:

- locally relevant to Portland
- clearly connected to the ad group's search intent
- understandable to a parent who does not know The Number Garden yet
- specific enough to avoid generic tutoring-marketplace language
- warm without becoming vague
- honest about the offer
- points toward the family visit when appropriate
- avoids implying test prep, speed, acceleration, or worksheet drilling as the main value

## Ad Groups

Build each ad group in the order Google shows the fields:

1. Ad group name
2. Final URL
3. Products or services to advertise
4. Keywords
5. Responsive search ad

The `products or services to advertise` field is a short theme/context field, not the actual keyword list. Use clean local phrases without quotes, brackets, or match-type syntax.

If Google generates loose suggestions like `math tutor for adults near me`, `tutoring centers near me`, or other low-fit terms, delete them and paste the planned keyword list below.

## First-Month Review Rhythm

Weekly:

- review search terms
- add negative keywords
- check which ad groups are spending
- check family-visit clicks and booked appointments
- note parent language from useful searches
- pause low-fit keywords

After 2-4 weeks:

- decide whether to add a second RSA per ad group
- decide whether `/family-visit/` or `/portland-math-tutor/` should be the main tutoring landing page
- decide whether exact/phrase volume is enough before testing broad match
