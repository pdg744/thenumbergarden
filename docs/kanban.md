# The Number Garden Kanban

This board is the working project-management document for The Number Garden.

It is intentionally simple. The goal is to help a solo founder keep momentum without building a fake enterprise process.

## How to use this board

- Move items between columns as work progresses.
- Keep `In Progress` to 1 to 3 items max.
- Break large vague tasks into concrete deliverables.
- Prefer tasks that end in a visible asset, decision, or conversation.
- Review the board at least twice per week.

## Definition of done

A task is done when one of these is true:

- copy is edited and live in the repo
- a reusable document or template exists
- a decision has been made and written down
- an outreach action has actually been taken
- a workflow is tested enough to use
